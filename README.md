# 🔐 Data Privacy Management System

## 📌 Description
A web-based application that allows users to manage their personal data privacy by requesting account deletion. The system includes user authentication, request tracking, and an admin panel for approval workflows.

## 🚀 Features
- User Registration & Login
- Secure Password Hashing
- Request Account Deletion
- View Request Status (Pending / Approved / Rejected)
- Admin Dashboard
- Approve / Reject Requests
- Delete Users

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS (Bootstrap)
